FEATURES
Aims at Full Automatic Function concurrently with Manual Operation, Lower Price and Compact type.
The machine composed of AC-servo motor reduces mechanical drive parts.
Heater Temperature controlled by Micro-Computer.

 Product Code Access automatically controls temperature loaded Micro-Computer.
Various kinds of Data such as bag counter, machine speed, bag length heater temperature, used bag film length and hour run meter are stored and displayed through micro-computer.
Nine(9) Kinds of Product Codes are Programmable

 Simply changing of product code allows bag length and Registration Mark Position to automatically set.
