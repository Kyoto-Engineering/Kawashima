KBF-6000XR High Speed Packing Machine

Max Speed 160bags/min.

High Speed continuous type Packaging Machine

 Applicable Bag size W80-200mm x L50-300mm (S)

 7 servo motors applied Model.


Feature
Absolute seal ability

 Two servo motors control for end seal.

 There are independent two servo motors.

 One is for End seal open/close. The other is End seal up/down.

 Optimal motion, stroke can be set on each bag size.

Multi Seal Motion can control Air volume of bag.

Seal pressure adjustable.

 There are one more setting beside of seal temperature, seal time.

 The third adjustment is Seal pressure.

Two servo motors control for center seal.

 There are independent two servo motors.

 One is for Center seal open/close.

 The other is Center seal up/down.

 Optimal motion, stroke can be set on each bag size.
Compact

 Machine is compact. Length of the packaging machine is only L1815mm
Easy Operation

User friendly touch sensitive screen.

Auto setting.

 Auto positioning of center sealer, photo sensor and date printer.

 This repeatable setting prevent human error, and minimizing machinery down time.
Minimum Loss

 Programable end seal motion can work as stripping mechanism and prevents product in seal.

Innovative Product Bite Detect Function can catch even smaller product in seal ever.

 This prevents fail bags go to markets.

Innovative Product Bite Detect Function is mounted on both end seal and center seal.

 This function can detect even seal wrinkle, seal failure both End seal and Center seal.
