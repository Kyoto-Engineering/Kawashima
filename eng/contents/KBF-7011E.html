FEATURES

With feature of Model KBF-7000E as is, this model is box-motion type machine integrated with Kawashima originally developed mechanism at end seal section for higher hermetic sealing condition of about 20% over our previous models.

 There are three types of standard, wide and high speed, depending on requirements.


SPECIFICATIONS
Standard type

 Operating speeds : 10 to 120 rpm

 Max. film web : 450mm

 Max. product dimensions : 200(W) x 90(H)mm

 Machine dimensions : 1,030(W) x 4,545(L) x 1,518(H)mm
Wide type

 Operating speeds : 10 to 100 rpm

 Max. film web : 650mm

 Max. product dimensions : 300(W) x 90(H)mm

 Machine dimensions : 1,180(W) x 4,545(L) x 1,518(H)mm
High speed type

 Operating speeds : 10 to 160 rpm

 Max. film web : 330mm

 Max. product dimensions : 140(W) x 50(H)mm

 Machine dimensions : 1,000(W) x 4,545(L) x 1,518(H)mm
Machine net weight : Approx. 1,300 to 1,400kg
Power consumption : 3 phase, 200V, about 5kW, 30A
Options :

No Product / No pack system, Auto Splicer, Date Printer, Auto feeder, etc.
