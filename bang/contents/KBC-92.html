FEATURES

This is automatic high speed box maker featuring that blank sheet is taken out one by one from magazine, formed into box and tape glue is applied to the bottom sealing.

 Hot melt glue seal type is also available.

 Compact design enables easy access for blank sheet supply, operation and maintenance from machine one side only and reduces space requirement.

 Microprocessor programable up to 99 datacodes allows for various kinds of box sizes and automatic size change.


SPECIFICATIONS
Operating speeds : 20 cases/min. (depending on case sizes)
Magazine capacity : 80 blank sheets (depending on thickness)
Power supply : 3 Phase, 200V, 10A, 2KVA
Air consumption : 165Nl/min., 500KPa (5kgf/sq.cm) at secondary side
Machine dimensions : 945(W) x 1,990(L) x 1,800(H)mm
Control/Display : Microcomputer control/LCD display
Machine weight : 430kg
Room temperature : 0 to 40degree C with humidity of 10 to 90%RH without dew
Tape to used : Craft tape, PP tape
Tape width : 38 to 50mm
Tape O.D. : Less than Dia.320mm
